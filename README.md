# Dasher

Create flexible dashboards with GitHub queries

## Development

**Install dependencies**

```shell
yarn
```

**Run frontend dev server**

```shell
cd frontend
yarn dev
```

**Run backend dev server**

```shell
cd backend
yarn dev
```

## Deployment

With [Now CLI](https://github.com/zeit/now-cli) installed globally, deploy the app by running:

```shell
now
```
